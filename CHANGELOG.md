## [v0.1.0] - 2026-01-11

### Highlights

- Cross-platform AI conversation sharing between ChatGPT, Claude Code, and other MCP clients
- Hugo-compatible storage for publishing conversations to static sites
- GitHub-based version-controlled conversation storage with automatic organization

### Added

- MCP server with stdio and HTTP transport support
- `save_conversation` tool - Save AI conversations with Hugo-compatible frontmatter
- `read_conversation` tool - Read conversations from storage
- `list_conversations` tool - List conversations with optional source filtering
- `search_conversations` tool - Search conversations by content
- `delete_conversation` tool - Delete conversations from storage
- `append_conversation` tool - Append content to existing conversations
- Hugo-compatible YAML frontmatter with standard and extension fields
- GitHub storage backend via omnistorage-github
- File organization by source platform (chatgpt/, claude/, gemini/, etc.)
- Automatic slug generation and date-based file naming
- Cross-platform conversation sharing between ChatGPT Desktop, Claude Code, and other MCP clients
