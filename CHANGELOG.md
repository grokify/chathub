## [v0.1.0] - 2026-01-11

### Highlights

- Cross-platform AI conversation sharing between ChatGPT, Claude Code, and other MCP clients
- Hugo-compatible storage for publishing conversations to static sites
- GitHub-based version-controlled conversation storage with automatic organization

### Added

- MCP server with stdio and HTTP transport support ([`8d4d384`](https://github.com/grokify/chathub/commit/8d4d384))
- `save_conversation` tool - Save AI conversations with Hugo-compatible frontmatter ([`8d4d384`](https://github.com/grokify/chathub/commit/8d4d384))
- `read_conversation` tool - Read conversations from storage ([`8d4d384`](https://github.com/grokify/chathub/commit/8d4d384))
- `list_conversations` tool - List conversations with optional source filtering ([`8d4d384`](https://github.com/grokify/chathub/commit/8d4d384))
- `search_conversations` tool - Search conversations by content ([`8d4d384`](https://github.com/grokify/chathub/commit/8d4d384))
- `delete_conversation` tool - Delete conversations from storage ([`8d4d384`](https://github.com/grokify/chathub/commit/8d4d384))
- `append_conversation` tool - Append content to existing conversations ([`8d4d384`](https://github.com/grokify/chathub/commit/8d4d384))
- Hugo-compatible YAML frontmatter with standard and extension fields ([`8d4d384`](https://github.com/grokify/chathub/commit/8d4d384))
- GitHub storage backend via omnistorage-github ([`8d4d384`](https://github.com/grokify/chathub/commit/8d4d384))
- File organization by source platform (chatgpt/, claude/, gemini/, etc.) ([`8d4d384`](https://github.com/grokify/chathub/commit/8d4d384))
- Automatic slug generation and date-based file naming ([`8d4d384`](https://github.com/grokify/chathub/commit/8d4d384))
- Cross-platform conversation sharing between ChatGPT Desktop, Claude Code, and other MCP clients ([`8d4d384`](https://github.com/grokify/chathub/commit/8d4d384))
